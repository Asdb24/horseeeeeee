# horses-racing-ticket-royal-racing

## How to run locally

1. Make sure you have Node.js installed.
2. From the project root, start the server:
   ```bash
   node server.js
   ```
3. Open your browser at `http://localhost:3000`.

If you only want to preview the static layout without API data, you can open `public/index.html` directly in a browser, but the live ticket data and countdowns require the local server.
